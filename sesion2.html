¿Que es la web?

La web Se designa como 'la web' al sistema de gestión de información más popular para la trasmisión de datos a través de internet. La web es el diminutivo de world wide web o www cuyas tecnologías para su funcionamiento (HTML, URL, HTTP) fueron desarrolladas en el año 1990 por Tim Berners Lee.
